# vue-todo

### 基于vue.js实现的TODO应用
---
本项目使用VS Code进行编写<br>
1.Star本项目<br>
2.Clone到本地<br>
3.运行npm install安装依赖包<br>
4.运行npm run build <br>
5.运行npm start<br>
即可在浏览器通过http://localhost:3333/ 访问应用页面<br><br>
本项目只实现了简单的登录功能，用户名和密码是写死的，具体见server/routers/user.js <br>
本项目使用线上数据库存储数据，若你对此感兴趣可访问https://www.apicloud.com/ 了解
